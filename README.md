# 🎮 AI Rock Paper Scissors Game

A real-time Rock-Paper-Scissors game using hand gesture recognition with OpenCV and MediaPipe.

## 🚀 Features
- Hand gesture detection (Rock, Paper, Scissors)
- Real-time webcam input
- Countdown timer
- Score tracking (Best of 3)
- Sound effects
- Interactive UI

## 🛠️ Technologies Used
- Python
- OpenCV
- MediaPipe
- NumPy

## ▶️ How to Run

```bash
pip install -r requirements.txt
python rps_game.py
